## Run Locally

Clone the project

```bash
  git clone https://https://github.com/mdimranh/ShopNow-Ecommerce-app.git
```

Go to the project directory

```bash
  cd ecomapp
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python manage.py runserver
```
Now click here
http://localhost:8000
